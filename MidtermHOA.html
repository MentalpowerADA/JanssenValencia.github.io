Midterm Examination

Welcome to the Midterm Exam: Ansible Infrastructure Automation! Your mission is to create a GitHub repository (CPE_MIDEXAM_SURNAME) and design a streamlined Ansible playbook, installing Elastic Stack, Nagios, Grafana, Prometheus, Influxdb, and Lamp Stack on separate hosts. Document each task with playbook codes and installation proofs. Include the GitHub push and commit process documentation. Paste your GitHub repository link for evaluation. Best of luck in showcasing your Infrastructure as Code skills with Ansible!
