Activity 12: Lab - Build a Sample Web App in a Docker Container

Welcome to Activity 12, a dynamic lab covering key scripting and containerization techniques. Begin by launching the DEVASC VM to set up the lab environment. Dive into basic bash scripting in Part 2, a prerequisite for subsequent activities. In Part 3, create and modify a Python script to craft a simple web application. Adjust the web app to use specific files in Part 4. Develop a bash script in Part 5 to automate Docker container creation, and finally, in Part 6, execute and verify the container using Docker commands. This lab provides a holistic exploration of scripting and Docker containerization, paving the way for advanced exploration in subsequent activities.

Prelim Period

Embark on a dynamic IT journey with a series of activities designed to sharpen your skills. In Activity 1, configure networks using Virtual Machines, bridging the gap between cloud computing and virtualization. Activity 2 advances to SSH Key-Based Authentication and Git setup for secure collaborative coding. Activity 3 focuses on installing an SSH server on CentOS or RHEL 8, a foundational step in secure server management. Elevate your command proficiency in Activity 4 by running elevated ad hoc commands, linking local and remote servers. Cap off your experience in Activity 5 by consolidating playbook plays, refining your Ansible automation skills. Each activity builds on the last, creating a dynamic and comprehensive learning curve for your IT toolkit. Let's dive in!
